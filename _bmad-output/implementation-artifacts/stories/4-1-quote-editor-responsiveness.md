# Story 4.1: Quote Editor Responsiveness

## Status: done

## Description
Le formulaire de création/édition de devis doit être utilisable sur mobile (375px).

## Acceptance Criteria
- [x] Quote creation form usable on 375px width (iPhone SE)
- [x] Item list scrollable with add/remove buttons accessible
- [x] Total calculation visible without scrolling
- [x] PDF preview responsive or shows "Download to preview" on mobile
- [x] Touch-friendly input fields (min 44px tap targets)
